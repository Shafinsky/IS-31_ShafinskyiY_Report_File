Медіа-запити

Для реалізації адаптивного дизайну сайту було використано медіа-запити, які дозволяють змінювати стилі в залежності від ширини екрану. Були визначені відповідні брейкпоінти, що забезпечують оптимальне розміщення елементів на сторінці для різних типів пристроїв.

Мобільна версія сайту

Цей медіа-запит застосовується для екранів з шириною 600px або менше. У цьому випадку, стилі включають:

  • Відображення навігаційного меню у вертикальному вигляді(з використанням кнопки)
  • Блокове вертикальне розташування елементів таблиці, що полегшує їх сприйняття на малих екранах.
  • Адаптація футера і контактної інформації для зручності користувача.

Код:

            @media (max-width: 600px) {
                .container {
                flex-direction: column;
                justify-content: flex-start;
                }
                nav.custom_navigation ul {
                display: none;
                }
                .toggle-button {
                display: block;
                }
                .news-grid {
                grid-template-columns: 1fr;
                }
                .statistics-grid {
                grid-template-columns: 1fr;
                }
                .rental-object {
                flex-direction: column;
                }
                .photo-gallery {
                height: 150px;
                }
                .info-block {
                min-width: 100%;
                }
                }

Планшетна/ноутбучна версія сайту

Цей медіа-запит застосовується для екранів з шириною від 600px до 1560px. Оператор and дозволяє поєднувати кілька умов у медіа-запиті. У запиті @media (min-width: 600px) and (max-width: 1560px) стилі будуть застосовуватися лише на пристроях, ширина яких знаходиться в межах від 600px до 1560px. Стилі включають:

  • Навігаційне меню відображається горизонтально, зручно для перегляду. Залежно від екрану (планшет або ноутбук) буде відображетися у 2 або 1 рядок.
  • Елементи таблиць отримують нові стилі, що покращують їх зовнішній вигляд.
  • Футер має більш ширше розташування, забезпечуючи доступ до важливих елементів.

Код:

            @media (min-width: 600px) and (max-width: 1200px) {
                .container {
                flex-direction: column;
                justify-content: center;
                }
                .news-grid {
                grid-template-columns: repeat(2, 1fr);
                }
                .statistics-grid {
                grid-template-columns: repeat(2, 1fr);
                }
                .rental-object {
                flex-direction: column;
                }
                .photo-gallery {
                height: 200px;
                }
                }

Десктопна версія сайту

Цей медіа-запит активується для екранів шириною понад 1560px. Логічний оператор not screen виключає застосування стилів для звичайних екранів(екранів шириною 1560px або менше).Основні зміни включають:

  • Центрування навігаційного меню.
  • Створення чітких контурів таблиць з акцентом на заголовки.
  • Футер розміщує елементи по краях, а не в центрі, що підвищує зручність використання.

Код:

            @media (min-width: 1200px) {
                .container {
                max-width: 1500px;
                justify-content: space-between;
                }
                .news-grid {
                grid-template-columns: repeat(3, 1fr);
                }
                .statistics-grid {
                grid-template-columns: repeat(4, 1fr);
                }
                .info-block {
                min-width: 300px;
                }
                .rental-object {
                flex-direction: row;
                }
                }
