Cелектори CSS

У даному розділі викладено теорія про види селекторів та загальні приклади їх застосування. Приклади, що стосуються індвидуального сайту розміщені у наступних розділах.

1. Селектор за тегом

Вибирає всі елементи певного типу (тегу).

p { color: blue; }

2. Селектор за класом

Вибирає всі елементи з певним класом.

.highlight { background-color: yellow; }

3. Селектор за ідентифікатором

Вибирає елемент з певним ідентифікатором.

#header { font-size: 24px; }

4. Груповий селектор

Вибирає кілька елементів одночасно.

h1, h2, h3 { font-family: Arial; }

5. Спадкові селектори

Вибирає елементи, які є нащадками певного елемента.

div p { color: green; }

6. Сусідні селектори

Вибирає елемент, який безпосередньо слідує за іншим елементом.

h1 + p { margin-top: 0; }

7. Селектори атрибутів

Вибирають елементи на основі значення їх атрибутів.

input[type="text"] { border: 1px solid black; }

8. Псевдокласи

Вибирають елементи на основі їх стану.

a:hover { color: red; }

9. Псевдоелементи

Вибирають частини елемента.

p::first-line { font-weight: bold; }
