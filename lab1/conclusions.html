Висновки

У цій лабораторній роботі ознайомилися з базовою структурою HTML-документа, яка складається з таких основних елементів, як <html>, <head>, <body>. Було розглянуто принципи вибору предметної галузі для веб-сторінки та використано різні HTML-елементи для створення інтерактивних та візуально привабливих документів. Також було створено шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.

Під час роботи з HTML-документом було досліджено такі елементи:

  • Посилання: створення гіперпосилань з використанням тегу <a> для навігації між сторінками.
  • Таблиці: додавання структурованих даних у вигляді таблиць з допомогою тегів <table>, <tr>, <td> та інших.
  • Зображення: вбудовування зображень на сторінку за допомогою тегу <img> для підвищення візуальної привабливості веб-сторінки.
  • Списки: створення впорядкованих і невпорядкованих списків за допомогою тегів <ul> та <ol>, що допомагає структурувати інформацію.
  • Форми: використання тегів <form>, <input>, <textarea>, <button> для збору даних від користувачів та взаємодії з ними.

Застосування цих елементів дозволяє створювати функціональні та інформативні веб-сторінки. Практична робота з HTML надала знання про основи роботи веб-документів, що є важливим етапом у процесі розробки сучасних веб-сайтів.
