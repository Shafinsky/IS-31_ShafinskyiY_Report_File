Висновки

Під час лабораторної роботи на тему "Каскадні таблиці стилів. Селектори. Ідентифікатори. Стильове оформлення текстових елементів в HTML-документах" ми придбали практичні навички, що дозволили нам поглибити розуміння основних концепцій веб-дизайну.

Під час роботи ми освоїли різні типи селекторів, такі як:

  • Типові селектори, які використовуються для вибору елементів за їхнім тегом.
  • Селектори класів, що дозволяють стилізувати групи елементів.
  • Ідентифікатори, які використовуються для специфічного стилізування окремих елементів.

Також ми розглянули різноманітні властивості, такі як колір, шрифт, розмір тексту та інтервали, які допомагають створити привабливий інтерфейс. Практичні завдання дозволили нам експериментувати з різними шрифтами, розмірами та кольорами, що сприяло кращому розумінню взаємодії стилів.

Особливу увагу ми приділили зовнішнім та внутрішнім відступам, які суттєво впливають на структуру та естетику веб-сторінки. Плаваючі елементи також були важливою частиною навчання, оскільки вони дозволяють створювати адаптивні макети, які добре реагують на різні розміри екранів.

Отже, завдяки цій лабораторній роботі ми отримали ґрунтовне розуміння основних принципів стилізації веб-документів. Ці знання є важливими для подальшого навчання у сфері веб-розробки, оскільки вони дозволяють створювати привабливі, функціональні та адаптивні веб-інтерфейси.

Контрольні питання

1. Для чого використовуються каскадні таблиці стилів (CSS)?

Каскадні таблиці стилів (CSS) використовуються для оформлення зовнішнього вигляду HTML-документів. Вони дозволяють змінювати кольори, шрифти, відступи, вирівнювання, розміри елементів, позиціювання та багато інших візуальних параметрів веб-сторінки.

2. Які методи визначення стилів в HTML-документі ви знаєте?

Існують три основні методи додавання стилів:

  • Inline-стилі: 
    <h1 style="color: red;">Заголовок</h1>
  • Внутрішні стилі: 
    <style>
h1 {
    color: blue;
}
</style>
  • Зовнішні стилі: 
    <link rel="stylesheet" href="styles.css">

3. У чому різниця між принципами каскадування і спадкування?

Каскадування — це процес вибору стилів із різних джерел і застосування тих, які мають вищий пріоритет. Спадкування — це механізм передачі стилів від батьківських елементів до дочірніх.

4. Як в HTML-документі оформлюються селектори тегу і клас?

Селектори використовуються для стилізації елементів:

  • Селектор тегу: 
    p {
    color: green;
}
  • Селектор класу: 
    .highlight {
    background-color: yellow;
}

5. Для чого використовується ідентифікатор?

Ідентифікатор використовується для унікальної ідентифікації елемента на сторінці. Він задається за допомогою атрибута id і використовується в CSS:

<div id="header">Заголовок</div>

#header {
    font-size: 24px;
    color: blue;
}

6. Навести приклади оформлення текстових елементів.

Приклади стилізації тексту за допомогою CSS:

  • Зміна кольору тексту: 
    p {
    color: red;
}
  • Зміна шрифту: 
    h1 {
    font-family: 'Arial', sans-serif;
}
  • Зміна розміру тексту: 
    h2 {
    font-size: 30px;
}
  • Застосування жирного шрифту: 
    strong {
    font-weight: bold;
}
  • Підкреслення тексту: 
    a {
    text-decoration: underline;
}
